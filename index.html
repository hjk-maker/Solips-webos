SETUP GUIDE

Get Running in 60 Seconds

No complicated setup. No package managers. Just download, extract, and open.

1
Download the Repository
Click the purple "Download ZIP" button above, or clone via Git if you prefer.
git clone https://github.com/hjk-maker/Solips-webos.git
2
Extract the Archive (if using ZIP)
Unzip the downloaded file to any folder on your computer.
3
Open the Entry File
Navigate into the extracted folder and open index.html in any modern browser (Chrome, Firefox, Edge, Safari).

Open Settings → AI Provider to connect Ollama, OpenAI, or other backends. Works fully offline without AI too.

FAQ

Frequently Asked Questions

Everything you need to know about Solips OS

What is a web operating system?
+
A web operating system is a fully functional OS that runs entirely in your web browser. It includes a file system, window manager, applications, and can even integrate AI - all without installing anything. Solips OS demonstrates this by using native browser APIs like IndexedDB for storage, DOM manipulation for windows, and BroadcastChannel for cross-tab communication.
Can you run an operating system in a browser?
+
Yes! Solips OS demonstrates this by using native browser APIs like IndexedDB for storage, DOM manipulation for windows, and BroadcastChannel for cross-tab communication. It's a real OS, not a simulation. You can create files, manage windows, run apps, and even control an AI agent - all within your browser.
What makes Solips OS the world's first?
+
Solips OS is the world's first zero-dependency web operating system with AI agent control. It uses only vanilla JavaScript and native browser APIs - no React, no Vue, no build tools. Every line of code does real work. The AI integration allows the LLM to control the OS through a virtual cursor and file system, making it a true AI-powered operating system.
How do I install Solips OS?
+
Download the ZIP from GitHub, extract it, and open index.html in any modern browser. No installation, no npm install, no build steps required. It just works. For the best experience with AI features, you can optionally run a local HTTP server, but it's not required for the core OS functionality.
Does Solips OS work offline?
+
Yes! The core OS works completely offline. All apps, file management, and window management work without an internet connection. AI features are optional - you can connect local models like Ollama or use cloud providers like OpenAI when online, but the OS itself doesn't require any network connectivity.
What AI providers does Solips OS support?
+
Solips OS supports multiple AI providers including Ollama (local), LM Studio (local), OpenAI, Anthropic Claude, Google Gemini, and Groq. Each provider has real, dedicated request handling - no fake unified wrappers. You can configure your preferred provider in Settings → AI Provider.
Is Solips OS really zero-dependency?
+
Absolutely. Zero npm packages. Zero frameworks. Zero build tools. The entire OS is written in vanilla JavaScript using only native browser APIs. The only external resources are fonts loaded from CDN (Inter and JetBrains Mono), which are optional - the OS works fine with system fonts.
Can I contribute to Solips OS?
+
Yes! Solips OS is open source under the MIT license. You can fork the repository on GitHub, make improvements, and submit pull requests. The codebase is well-organized and documented, making it easy to understand and contribute to. Check out the GitHub repository for contribution guidelines.
